---
title: "How to install Entity Framework 4.1 RC?"
date: "2011-09-23"
categories: 
  - "entity-framework"
  - "general"
---

You need to open the solution where you want to have the latest version of EF. If you are running under .Net 3.5, big chance that you are running on the version 1. If you are running on .Net 4.0 you should be on the EF version 4. The 4.1 require to do some manual change.

First, be sure you have the NuGet extension.

\[caption id="attachment\_285" align="aligncenter" width="400" caption="NuGet package installed"\][![](http://patrickdesjardins.com/blog/wp-content/uploads/2011/09/nugetpackage-400x90.png "nugetpackage")](http://patrickdesjardins.com/blog/how-to-install-entity-framework-4-1-rc/nugetpackage)\[/caption\]

Second, open the console of NuGet and type : install-package EntityFramework. To open the console you need to go to View > Other Windows > Package Manager Console.

\[caption id="attachment\_288" align="aligncenter" width="400" caption="Installation of Entity Framework 4.1 RC"\][![](http://patrickdesjardins.com/blog/wp-content/uploads/2011/09/installef41-400x123.png "installef41")](http://patrickdesjardins.com/blog/how-to-install-entity-framework-4-1-rc/installef41)\[/caption\]

EntityFramework 4.1 will be installed on the solution that is active.

You can also use the NuGet Package Manager which give you a visual interface to download Entity Framework. This is located in Project > Manage NuGet Package.

[![](http://patrickdesjardins.com/blog/wp-content/uploads/2011/09/entityFrameworkByNuGetManager-400x267.png "entityFrameworkByNuGetManager")](http://patrickdesjardins.com/blog/how-to-install-entity-framework-4-1-rc/entityframeworkbynugetmanager)

Once done, your project will now have a reference to the DLL of Entity Framework. [![](http://patrickdesjardins.com/blog/wp-content/uploads/2011/09/entityFrameworkByNuGetReference.png "entityFrameworkByNuGetReference")](http://patrickdesjardins.com/blog/how-to-install-entity-framework-4-1-rc/entityframeworkbynugetreference)
