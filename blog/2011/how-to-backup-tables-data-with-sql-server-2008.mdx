Some time it can be useful to dump all data of a table in the format of SQL insert statement.

This can be done easily with _Microsoft SQL Management Studio_. The first step is to right click on the database and select the option **Task>Generate Script**. This will popup a wizard window.

\[caption id="attachment\_641" align="aligncenter" width="400" caption="Microsoft Sql Management Studio Generate scripts"\][![](http://patrickdesjardins.com/blog/wp-content/uploads/2011/12/Snap-2011-12-13-at-14.33.57-400x349.png "Snap 2011-12-13 at 14.33.57")](http://patrickdesjardins.com/blog/how-to-backup-tables-data-with-sql-server-2008/snap-2011-12-13-at-14-33-57)\[/caption\]

The last step is to select that we desire only the data.

\[caption id="attachment\_642" align="aligncenter" width="400" caption="Select data to get all information of the table without creating the schema"\][![](http://patrickdesjardins.com/blog/wp-content/uploads/2011/12/Snap-2011-12-13-at-14.36.05-400x288.png "Snap 2011-12-13 at 14.36.05")](http://patrickdesjardins.com/blog/how-to-backup-tables-data-with-sql-server-2008/snap-2011-12-13-at-14-36-05)\[/caption\]

This will produce all SQL's insert statement with your data.
