**MSDTC is unavailable** is an error that raise when the service "**Distributed Transaction Coordinator**" is not running.

What you have to do is to run the service. When you boot your computer (or your server) it's normally by default with Startup Type "Manual".

[![msdtcServiceList](http://patrickdesjardins.com/blog/wp-content/uploads/2013/03/msdtcServiceList1-400x154.png)](http://patrickdesjardins.com/blog/?attachment_id=1960)

What you have to do is to select the "Automatic" startup type.

[![msdtcStatupAutomatic](http://patrickdesjardins.com/blog/wp-content/uploads/2013/03/msdtcStatupAutomatic-350x400.png)](http://patrickdesjardins.com/blog/?attachment_id=1961)
