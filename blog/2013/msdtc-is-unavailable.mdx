**MSDTC is unavailable** is an error that raise when the service "**Distributed Transaction Coordinator**" is not running.

What you have to do is to run the service. When you boot your computer (or your server) it's normally by default with Startup Type "Manual".

![](images/msdtcServiceList1-400x154.png)

What you have to do is to select the "Automatic" startup type.

![](images/msdtcStatupAutomatic-350x400.png)
